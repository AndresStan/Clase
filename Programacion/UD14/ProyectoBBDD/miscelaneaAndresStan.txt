-- CONSULTAS CON OPERADORES DE CONJUNTO: UNION, INTERSECT, MINUS


1. CodEmpDir de empleados que dirigen algún departamento con los que dirigen
algún centro. Sin duplicados.
select d.CodEmpDir from departamento d union  select c.CodEmpDir from centro c

2. Igual que la anterior consulta pero con UNION ALL para ver también los duplicados
(directivos de depto y centro a la vez).
select d.CodEmpDir from departamento d union all select c.CodEmpDir from centro c

3. CodDep y NomDep de departamentos con PreAnu > 20.000.000 y los
departamentos con TiDir = 'F'. Sin duplicados.
select CodDep, NomDep from departamento where PreAnu > 20000000 union select CodDep, NomDep
from departamento where TiDir = 'F';

4. Simula INTERSECT: empleados que dirigen un departamento y también dirigen un
centro. Usa IN.
select c.CodEmp, c.NomEmp from empleado c where c.CodEmp in (select CodEmpDir from departamento) 
intersect
select e.CodEmp, e.NomEmp from empleado e where e.CodEmp in (select CodEmpDir from centro)

5. Simula EXCEPT: empleados que dirigen un departamento, pero NO un centro. Usa
NOT IN y también LEFT JOIN … IS NULL.
select d.CodEmpDir from departamento d
except 
select c.CodEmpDir from centro c


6. DesHab de habilidades que tienen empleados de 'PROZS' y las que tienen
empleados de 'VENZS'. Sin duplicados.
select DesHab from habilidad where CodHab in (select CodHab from habemp where CodEmp in (select CodEmp from empleado where CodDep='PROZS'))
union 
select DesHab from habilidad where CodHab in (select CodHab from habemp where CodEmp in (select CodEmp from empleado where CodDep='VENZS'))

7. Simula INTERSECT: habilidades que poseen empleados TANTO de 'PROZS' COMO de
'VENZS'. Usa subconsulta con in
select DesHab from habilidad where CodHab in (select CodHab from habemp where CodEmp in (select CodEmp from empleado where CodDep='PROZS'))
intersect
select DesHab from habilidad where CodHab in (select CodHab from habemp where CodEmp in (select CodEmp from empleado where CodDep='VENZS'))
